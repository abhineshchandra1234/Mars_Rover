# Mars Rover
## This rover fetches a list of rovers that went to Mars. It fetches a list of their mission and another list of photos taken in that mission.
## Tech Stack
- MVVM
- Jetpack Compose
- Navigation Compose
- Retrofit
- Dagger Hilt
- Lifecycle Compose
- Coil Compose
- Room
- Google font
## API Used
- Free NASA API used in the project
- [NASA API](https://api.nasa.gov/)
